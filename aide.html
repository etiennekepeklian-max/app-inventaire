Centre d'aide

💡 Astuce : Cliquez sur le logo "Carrés entrelacés" à côté d'une référence pour la copier instantanément !

➕ Ajouter / Corriger un objet

Ce module permet d'enregistrer de nouveaux objets dans l'inventaire.

  • Référence auto : Tapez le nom de l'objet, la référence (ex: TAB-1234) se crée toute seule.
  • Catégories : Sélectionnez la catégorie dans le menu déroulant dynamique.
  • Notes : Vous pouvez ajouter un commentaire (état de l'objet, consigne spéciale).
  • 🖨️ Une fois validé, vous pouvez imprimer l'étiquette QR Code directement depuis votre mobile.
  • ✏️ Corriger : Si vous avez fait une erreur de frappe, cliquez sur "Corriger une erreur" après validation, modifiez le texte et ré-enregistrez pour écraser la faute.
🔍 Rechercher et Scanner

Ouvrez la fiche complète d'un objet en flashant son QR Code ou en tapant une partie de son nom/référence.

  • Détails : Affiche l'emplacement, la catégorie, le créateur et le statut.
  • Ajouter une note : Insérez une remarque datée et signée à votre nom directement depuis la fiche.
  • Actions : Depuis cette fiche, vous pouvez Déplacer ou Sortir l'objet.
  • ↩️ Réintégrer : Si vous scannez un objet qui était "Sorti", un bouton vous permettra de le remettre instantanément en stock actif.
🚀 Le Mode Rafale (Scan multiple)

Disponible dans la zone "Rechercher et Scanner", ce mode permet de gagner énormément de temps.

  • Scannez vos objets à la chaîne. La caméra ne se coupe pas et la liste s'affiche en direct.
  • Une fois terminé, validez pour déplacer ou archiver tout ce lot d'objets en un seul clic.
📋 Inventaire et Historique

Consultez la base de données complète et synchronisée.

  • Tri & Recherche : Utilisez les boutons bleus pour trier par Nom, Emplacement, ou Date. La barre de recherche filtre instantanément les résultats.
  • Fiche au clic : Touchez n'importe quelle carte de l'inventaire pour ouvrir la fiche détaillée de l'objet.
  • 🕒 L'Historique (Admin) : En haut à droite, un bouton permet de voir le journal de bord de toutes les actions de l'équipe (Ajouts, Sorties, Déplacements).
📦 L'Audit de Zone

Outil de pointage très puissant pour vérifier qu'aucun objet ne manque dans une réserve. Il fonctionne même sans réseau !

  • 1. Saisissez le nom de l'emplacement (ex: Réserve B).
  • 2. Flashez absolument tous les objets présents devant vous.
  • 3. Le Rapport : L'application calcule ce qui est validé (🟢), ce qu'il manque (🔴), et les objets qui sont physiquement là mais qui devraient être ailleurs (🟡).
⚙️ L'Administration (Réservé Admin)

Accessible uniquement aux responsables de l'application :

  • 📊 Tableau de bord : Graphique de répartition du stock.
  • 🏷️ Gérer les catégories : Ajoutez de nouvelles catégories ou supprimez-en (elles apparaîtront dans les menus déroulants de tout le monde).
  • 📥 Import / Export : Téléchargez l'inventaire en CSV, ou importez jusqu'à 500 objets d'un coup.
  • 👥 L'équipe : Créez de nouveaux accès, modifiez les mots de passe de votre équipe, ou supprimez des comptes.
🔒 Sécurité, Cache et Mode Hors-ligne

L'application est conçue pour fonctionner dans toutes les conditions :

  • Hors-Ligne : Si vous n'avez pas de réseau, vous pouvez tout de même consulter l'inventaire, rechercher des objets, et effectuer un Audit de Zone.
  • Vider le cache : Dans "Mon Profil", ce bouton rouge force l'application à se nettoyer si vous constatez un bug d'affichage.
  • ⏱️ Sécurité 30s : Si vous changez d'application ou verrouillez votre téléphone plus de 30 secondes, l'application vous déconnecte automatiquement pour protéger vos données.
