Centre d'aide

💡 Astuce : Cliquez sur le petit logo à droite d'une référence pour la copier instantanément !

🔍 Rechercher et Scanner

Ouvrez la fiche complète d'un objet en flashant son QR Code ou en tapant une partie de son nom/référence.

  • Détails : Affiche l'emplacement, la catégorie, le créateur, les notes et le statut.
  • Ajouter une note : Insérez une remarque datée et signée à votre nom directement depuis la fiche.
  • Actions : Depuis cette fiche, vous pouvez Déplacer ou Sortir l'objet.
  • ↩️ Réintégrer : Si vous scannez un objet qui était "Sorti", un bouton vous permettra de le remettre instantanément en stock actif.
➕ Ajouter / Corriger un objet

Ce module permet d'enregistrer de nouveaux objets dans l'inventaire.

  • Référence auto : Tapez le nom de l'objet, la référence se génère toute seule. Vous pouvez la modifier si besoin.
  • Catégories : Sélectionnez la catégorie dans le menu déroulant (trié de A à Z).
  • Notes : Vous pouvez ajouter un commentaire (état, remarque particulière).
  • 🖨️ Une fois validé, imprimez l'étiquette QR Code directement depuis votre mobile.
  • ✏️ Corriger : Si vous avez fait une erreur de frappe, cliquez sur "Corriger une erreur" après validation, modifiez le texte et ré-enregistrez pour écraser la faute.
🔄 Déplacer un article

Pour modifier l'emplacement d'un objet existant.

  • Flashez le QR code ou saisissez manuellement sa référence.
  • Indiquez sa nouvelle destination (ex: Camionnette B).
📤 Sortir un article

Pour archiver un objet qui quitte le stock (Vente, Restauration, Destruction...).

  • Indiquez obligatoirement la raison (le motif) pour valider la sortie. Il sera conservé dans l'historique et sur la fiche de l'objet.
📋 Inventaire et Historique

Consultez la base de données complète, triée par défaut du plus récent au plus ancien.

  • Tri & Recherche : Utilisez les boutons bleus pour trier par Nom, Catégorie ou Lieu. La barre de recherche filtre instantanément la liste.
  • Fiche au clic : Touchez n'importe quelle carte pour ouvrir la fiche détaillée.
  • 🖨️ Réimprimer : Depuis la fiche d'un objet en inventaire, vous pouvez réimprimer son QR Code à tout moment.
  • 🕒 L'Historique (Admin) : L'icône en haut à droite permet de voir le journal de bord de toutes les actions de l'équipe.
🚀 Le Mode Rafale (Scan multiple)

Disponible dans la zone "Rechercher et Scanner", ce mode permet de traiter des lots d'objets.

  • Scannez vos objets à la chaîne. La caméra reste allumée et mémorise la liste.
  • Une fois terminé, choisissez de déplacer ou d'archiver tout le lot d'objets en un seul clic.
📦 Faire un Audit

Outil de pointage puissant pour vérifier votre stock. Synchronisé en temps réel avec la base de données.

  • Audit de zone : Spécifiez une réserve et scannez ce qui s'y trouve.
  • Audit Total : Scannez tous les objets de l'entreprise sans distinction de pièces.
  • 🛠️ Outils d'audit : Si un QR est illisible, utilisez la Saisie Manuelle. Vous pouvez aussi Mettre en pause votre audit pour le reprendre plus tard, ou utiliser la Fusion Collaborative pour assembler vos scans avec ceux d'un collègue.
  • Le Rapport : L'application calcule les objets confirmés (🟢), manquants (🔴), et inconnus ou mal rangés (🟡).
  • 🖨️ Vous pouvez Imprimer le rapport détaillé à la fin de l'audit pour conserver une trace des anomalies et des présences.
⚙️ L'Administration (Réservé Admin)

Espace de gestion réservé aux responsables de l'application :

  • 🏷️ Gérer les catégories : Ajoutez de nouvelles catégories ou supprimez-en avec la croix rouge.
  • 📥 Import de fichier CSV : Chargez un fichier pour importer jusqu'à 500 objets. L'outil est intelligent : si une référence du fichier existe déjà dans l'application, elle sera mise à jour au lieu d'être créée en double.
  • 📤 Export CSV : Téléchargez l'inventaire formaté en 11 colonnes, structuré exactement comme la base de données.
  • 👥 L'équipe : Créez de nouveaux accès, modifiez les mots de passe, ou supprimez définitivement des comptes.
👤 Mon Profil & Sécurité

Accessible via l'icône Profil sur la page d'accueil :

  • Mot de passe : Modifiez votre clé d'accès personnelle.
  • Vider le cache : Un bouton de maintenance pour forcer la mise à jour visuelle si l'application rencontre un bug d'affichage.
  • ⏱️ Sécurité d'arrière-plan : Si l'application reste cachée ou que votre téléphone est verrouillé pendant plus de 30 secondes, l'accès se bloque automatiquement pour protéger vos données.
