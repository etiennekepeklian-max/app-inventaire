Mode d'emploi

🔍 Recherche et Inventaire

Ce module vous permet de localiser rapidement un objet et de consulter sa fiche.

  • Scanner un code : Utilisez l'appareil photo pour scanner un QR code généré par l'application ou un code-barres commercial standard.
  • Recherche textuelle : Tapez le nom, la catégorie ou la référence d'un objet.
  • Historique : Sur la fiche d'un objet, cliquez sur "Historique des mouvements" pour voir le journal détaillé des actions liées à cet article.
➕ Ajouter ou Corriger un objet

Utilisez ce module pour faire entrer un nouvel objet dans le stock ou corriger une erreur.

  • Création rapide : Tapez un nom long (plus de 3 lettres), et une référence unique sera générée automatiquement.
  • Associer un code existant : Cliquez sur l'icône 📷 pour scanner le code-barres original du produit (EAN) et le lier à la fiche.
  • Correction : Si vous avez fait une erreur en ajoutant un objet, modifiez les champs et cliquez à nouveau sur "Mettre à jour la fiche".
  • Impression : Une fois validé, vous pouvez imprimer une étiquette propre contenant le QR code.
🔄 Déplacer et 📤 Sortir

Ces modules permettent de tracer les flux de la marchandise.

  • Déplacement individuel : Scannez un objet et sélectionnez sa nouvelle zone dans le menu déroulant.
  • Mode Rafale 🚀 : Dans le module de Scan, basculez en "Mode Rafale". Scannez plusieurs objets à la suite pour les déplacer ou les sortir simultanément. (Une alerte se déclenchera si vous scannez un objet déjà sorti).
  • Archivage (Sortie) : Un motif est obligatoire (Vendu, Détruit, etc.). L'objet disparaîtra du stock actif mais restera conservé dans l'historique global de la base.
📦 Lots, Kits et Audits
  • Assembler des Kits : Sur la fiche descriptive d'un objet (Parent), ouvrez la section "Mode Lot" pour y associer d'autres sous-références.
  • Faire un Audit : Choisissez une zone spécifique. Le système calculera les objets attendus. Scannez tout ce que vous voyez. Le rapport final (imprimable) croisera vos scans avec la base pour lister les manquants et les anomalies (ex: un objet prévu ailleurs).
⚙️ Administration du système

Outils exclusifs au rôle Administrateur.

  • Tableau de bord : Suivez les statistiques, les alertes d'anomalies, les objets dormants (+6 mois) et l'activité de l'équipe du jour.
  • Structure (Zones/Catégories) : Ajoutez ou supprimez les emplacements officiels pour éviter les fautes de frappe sur le terrain.
  • Import/Export CSV : Téléchargez la sauvegarde complète ou importez un tableau Excel. Le système est intelligent : si une référence existe déjà, il mettra à jour sa fiche sans créer de doublon.
  • Équipe : Gérez les mots de passe et les droits (Admin, Standard, ou Lecteur).
  • Diagnostics : Vérifiez l'état de la connexion avec le serveur Google et de la caméra de l'appareil.
  • Historique Global : Accessible via l'icône 🕒 dans l'inventaire, permet de voir l'intégralité du registre et de le filtrer (Par action, date, utilisateur...).
